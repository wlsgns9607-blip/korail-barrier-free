글자 대신 그림으로,
침묵 대신 목소리로 안내합니다.

대중교통 이용객의 절반 이상이 40대 이상입니다. 그런데도 코레일톡은 여전히 작은 글자와 좁은 버튼으로 말합니다. 이 기획은 모바일 화면, 승강장 모니터, 열차 내부를 하나의 배리어프리 경험으로 잇습니다.

ISSUE 01

주요 버튼이 작아 터치 미스가 잦고, 눈이 침침한 이용객은 조작 자체가 어렵습니다.

ISSUE 02

좌석 표기가 "5호차 11A" 같은 텍스트뿐이라, 문을 열고도 자리를 못 찾고 헤맵니다.

ISSUE 03

승무원 인력이 부족해 승차권 확인이나 불편사항을 물어봐도 적시에 응답받지 못합니다.

ISSUE 04

고령층 이용 비중은 계속 늘고 있지만, 앱과 열차 환경은 그 속도를 따라가지 못합니다.

Before → After

같은 화면, 다른 경험

버튼 크기와 좌석 안내 방식만 바꿔도, 승객이 자리를 찾는 데 걸리는 시간과 터치 성공률이 달라집니다.

Core Solution

화면 하나로 끝나지 않는 기획

모바일 UX, 승강장 인프라, 열차 내부까지 — 세 개의 접점을 하나의 배리어프리 여정으로 설계했습니다.

FEATURE 01 · MOBILE UX

들리는 배리어프리 모드

실버 모드를 켜면 주요 버튼이 화면 절반 크기로 커집니다. "내 자리 어디야?"라고 물으면 AI가 스피커로 "오른쪽 통로로 3미터, 창가 쪽입니다"라고 안내합니다.

Web Speech API
FEATURE 02 · PLATFORM

열차 진입 전, 승강장 광고 모니터에 다가오는 호차의 좌석 배치도와 문 방향을 크게 띄웁니다. 타기 전에 동선을 알면 탑승 병목이 사라집니다.
